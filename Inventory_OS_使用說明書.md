# 庫存分析智能儀表板 (Inventory Strategic OS) - 使用說明書

這是一套功能強大且完全運行在您本機瀏覽器中的「互動式庫存分析系統」。它無需安裝複雜的後端環境或傳統的 BI 軟體，只需透過瀏覽器打開 HTML 檔案即可享受深色模式與即時數據圖表的高階分析體驗。

---

## 快速啟動指南

1.  **開啟系統**：請用最新版的 Google Chrome 或 Microsoft Edge 雙擊打開 `Inventory_Strategic_OS.html`。
2.  **上傳您的資料**：點選導覽列最右側的 **「設定與資源」** 頁籤。
3.  點擊網頁中的「框線拖曳區」來上傳您的 Excel (.xlsx 或 .csv) 檔案。如果您還沒有資料，您可以點擊「**載入測試範例資料**」來檢視系統渲染效果。
4.  上船後系統會自動運算並提示「載入並分析完成！」這時您可以切換至各個頁籤查看圖表。

---

## 支援的資料格式 (Data Schema)

為確保系統能正確解析、計算並繪製圖表，請確保您的 Excel 或 CSV 檔案中包含以下**必要欄位 (需中英文一致或至少以相同的英文標題命名)**：

| 欄位名稱 (Header) | 說明 | 範例資料 |
| :--- | :--- | :--- |
| `Date` | 紀錄日期 (建議格式 yyyy-mm-dd) | 2026-03-01 |
| `ProductID` | 產品的唯一識別碼 | P1001 |
| `ProductName` | 產品易讀名稱 | Electronics Model 0 |
| `Category` | 產品分類 | Electronics |
| `In_Qty` | 當日/該期間的入庫數量 | 150 |
| `Out_Qty` | 當日/該期間的銷售或出庫數量 | 12 |
| `Current_Inventory` | 當日/該期間的期末剩餘庫存量 | 300 |
| `UnitCost` | 平均單位成本 (計算存貨價值與COGS用) | 250.5 |
| `UnitPrice` | 銷售單價 (計算營收與存銷比用) | 400.0 |
| `Average_Aging_Days`| 該品項的系統預估平均庫齡 (天) | 45 |
| `LeadTime_Days` | 該品項採購前置作業時間 (天) | 14 |

*(註：如果您使用系統根目錄中的 `generate_sample_data.py`，它會自動產生符合上述格式的 `Sample_Inventory_Data.xlsx` 供您測試使用。)*

## 核心頁籤功能解說

### 1. 總覽儀表板 (Overview)
提供最重要的四大 KPI：總庫存金額（估值）、平均庫存天數 (DSI)、加權庫齡、以及系統自動評分的健康分數。並有一目了然的存貨價值趨勢與品類佔比圖。

### 2. 趨勢與周轉 (Trend & Turnover)
此區深入到時間維度。您將看到每個月的「庫存天數 (折線)」對抗「存貨周轉率 (柱狀)」的拉扯。
*   目標：縮短 DSI，提升存貨周轉率。
*   下方的存銷比 (Stock-to-Sales Ratio) 指數可看出資金被庫存綁住的嚴重程度。

### 3. 結構與 ABC (Structure & ABC)
系統會自動彙整庫存金額佔比並採用累計比例法執行 **A (前50%)、B (50~70%)、C (70%以外)** 的分級。
在右側的散佈圖中，可以比對「庫存佔比 (投資資源)」與「銷售佔比 (回報需求)」，如果點落在靠近對角線為理想狀態，過度偏移則代表可能有缺貨危機或資金呆滯。

### 4. 庫齡與風險 (Aging & Risk)
分析不同落點的庫齡佔比（<30天 ~ >120天），系統同時內建一套「減損提列邏輯」，只要庫齡大於 60 天即會依據天數提列漸進的呆滯損失，您可以在儀表板右下角即時看到如果清算可能面臨的「預估呆滯減損風險」。

### 5. 預測與安全庫存 (Forecast & ROP)
自動依照 (日均銷量 * 前置時間) + (安全庫存計算公式) 得出 ROP 點。並自動過濾出「當前庫存」已經跌破「ROP 再訂購點」的危險品項，供您作為補貨報表處理。

### 6. 品項深度剖析 (Item Deep Dive)
若您需要看細部數據，此頁面提供完整的大數據清單，包含售磬率 (STR)、ABC 等級、金額等，並且支援即時中文輸入搜尋，能立即過濾出所需產品資訊。

### 7. 策略行動 (Strategic Actions)
AI 行動決策區。系統根據全盤資料檢驗後，會給出白話的行動建議（例如：A 類產品的缺貨警告、DSI 過高的警告、C 類庫存放置過久應安排出清等），是您快速匯報與開會的最強輔助工具。

---

如果遇到圖表無法讀取，請按 `F5` 重新整理頁面，並重新上傳您的 Excel 檔案即可。儀表板支援匯出 JSON 功能可將計算完成的狀態轉存下來。
