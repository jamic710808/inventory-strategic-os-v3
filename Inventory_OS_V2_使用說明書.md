# 庫存分析智能儀表板 V2.0 - 使用說明書

這是一套功能強大且完全運行在瀏覽器中的「互動式庫存分析系統」。無需安裝後端環境或 BI 軟體，只需開啟 HTML 檔案即可享受深色模式、即時計算與動態圖表的專業分析體驗。

---

## 快速啟動

1. **開啟系統**：用 Chrome 或 Edge 開啟 `Inventory_Strategic_OS_V2.html`
2. **載入資料**：點選最右側的 **「設定DAX」** 分頁
3. **選擇方式**：
   - 拖曳 Excel 檔至上傳區，或點擊上傳區選擇檔案
   - 或點擊「**載入測試範例資料**」快速預覽
4. 系統自動運算後彈出「載入並分析完成！」，即可切換各分頁查看

---

## 支援的資料格式

Excel 或 CSV 檔案需包含以下欄位：

| 欄位名稱 | 說明 | 範例 |
|---------|------|------|
| `Date` | 日期 (yyyy-mm-dd) | 2026-03-01 |
| `ProductID` | 產品唯一碼 | P1001 |
| `ProductName` | 產品名稱 | Electronics Model 0 |
| `Category` | 分類 | Electronics |
| `In_Qty` | 入庫數量 | 150 |
| `Out_Qty` | 出庫/銷售數量 | 12 |
| `Current_Inventory` | 期末庫存量 | 300 |
| `UnitCost` | 平均單位成本 | 250.5 |
| `UnitPrice` | 銷售單價 | 400.0 |
| `Average_Aging_Days` | 平均庫齡 (天) | 45 |
| `LeadTime_Days` | 採購前置時間 (天) | 14 |

## 九大分頁功能指南

### 1. 總覽儀表板
8 大 KPI 一目了然：總庫存金額、DSI、庫齡、健康分數、周轉率、售磬率、GMROI、缺貨品項。含庫存趨勢折線圖與品類甜甜圈圖。

### 2. 趨勢與周轉
- **周轉率/DSI 雙軸圖**：目標是提升柱狀（周轉率）、降低折線（DSI）
- **存銷比 SSR**：越低代表資金周轉越快
- **月度成長率**：綠色=成長、紅色=衰退
- **移動平均**：3 個月 / 6 個月平滑趨勢

### 3. 結構與 ABC
- **切換按鈕**：相對法（累計比例）/ 絕對法（固定門檻）
- **供需失衡矩陣**：點在對角線上為理想，偏離代表過多或不足
- **品類×ABC 矩陣表**：掌握各品類的 ABC 組成

### 4. 庫齡與風險
- **庫齡五級分佈**：<30天 → >120天，顏色漸深代表風險漸高
- **呆滯減損預估**：自動計算 >60 天庫存的潛在減損金額
- **背離信號**：偵測售磬率↑但庫齡也↑的矛盾情形
- **健康評分卡**：四維雷達圖綜合評估供應鏈狀態

### 5. 預測與安全庫存
- **季節指數雷達**：看出哪幾個月是旺季/淡季
- **Fisher 檢驗**：判斷季節性是否統計顯著
- **信心區間帶**：預測值 ±2σ 的可信範圍
- **ROP 預警表**：列出庫存低於再訂購點的品項

### 6. 供應鏈效率 ★新功能
- **GMROI**：每1元庫存投資帶來多少毛利
- **缺貨成本估算**：基於日均銷量估算的潛在損失
- **供需熱力圖**：品類×月份的銷量強度色彩圖
- **補貨排程表**：建議補貨量、預計耗盡日、優先度

### 7. 品項深度剖析
- 支援**搜尋**（輸入關鍵字即時過濾）
- 支援**品類篩選**和 **ABC 等級篩選**
- 點擊欄頭可**排序**（品類、ID、金額、庫齡等）
- 右下角有**分頁導覽**
- 可**匯出 CSV** 供進一步分析

### 8. 策略行動
AI 智能診斷自動偵測 5 大風險情境並給出行動建議：
- 🔴 A 類缺貨危機
- 🟡 C 類呆滯警示
- 🟠 低 GMROI 警示
- 🟣 供需失衡
- 🔵 周轉過慢

右側 KPI 評分卡彙整六維分數。

### 9. 設定與 DAX 參考
- **上傳功能**：支援拖放 + 點擊
- **匯出功能**：JSON（完整分析結果）/ CSV（品項清單）
- **重置**：清除所有資料回復初始狀態
- **DAX 公式庫**：12 條核心公式的說明與計算邏輯

## 常見問題

**Q: 圖表沒有出現？**
按 F5 重新載入頁面，重新上傳 Excel。確認網路連線正常（需載入 CDN 資源）。

**Q: 資料量上限？**
瀏覽器端建議不超過 5 萬筆交易紀錄，通常 1~3 萬筆流暢無延遲。

**Q: 可以使用現有的 Sample_Inventory_Data.xlsx 嗎？**
可以，該檔案的欄位結構與系統完全相容。
